## Level 1. 문제 5

```javascript
// 입력
solution(2, 5)
solution(-4, 2)

// 출력
[2, 4, 6, 8, 10]
[-4, -8]
```

```javascript
// 내가 푼 답
function solution(x, n) {
    var answer = [];
    var a = x;
    
    for(let i=0; i<n; i++){
        answer.push(x);
        x += a;
    }
    
    return answer;
}
```
