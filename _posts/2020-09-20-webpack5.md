## npm 초기화 명령어 init

```bash
npm init
```

위 명령어를 실행하면 `package.json` 파일이 생성된다.  
하지만 터미널 창에서 질문을 엄청한다.  
이런 질문이 귀찮으시다면 뒤에 `-y` 옵션값을 붙여서 실행하시면 된다.

```bash
npm init -y
```

위와 같이 명령어를 실행하면 `package.json` 파일이 바로 생성된다.
