## 16.1.1 고정 소수점

소수점 뒤 자릿수를 지정하는 형식을 원한다면 `toFiexed()`를 사용합니다.

```javascript
const x = 19.51;
x.toFixed(3);       // "19.510"
x.toFixed(2);       // "19.51"
x.toFixed(1);       // "19.5"
x.toFixed(0);       // "20"
```

![](/static/img/learningjs/image149.jpg)

이 숫자는 버림(truncation)이 아니라 반올림(round)입니다.
