## npm 설치 명령어 - install

```bash
npm install jquery
```

위 명령어를 실행하면 해당 명령어를 실행한 위치에 `node_modules` 폴더가 생기면서 해당 폴더 안에 `jquery` 라이브러리가 설치된다.  
`node_modules/jquery` 이런식으로 말이다.

그리고 자동으로 `package.json`에 jquery 버전이 명시된다.
